# JohnnyMart Website

Static JohnnyMart starter website for GitHub Pages.

## Files
- `index.html` — public website
- `styles.css` — design/theme
- `products.js` — starter product data and categories
- `script.js` — storefront behavior
- `admin.html` — simple local inventory helper
- `admin.js` — admin page behavior
- `assets/` — images

## Important
The admin page saves new products only in your current browser. To make products permanent and public, copy the exported product data from `admin.html` into `products.js`, then commit the change to GitHub.

## GitHub Pages
1. Upload all files in this folder into your JohnnyMart repository.
2. Go to repository Settings → Pages.
3. Under Source, choose `Deploy from a branch`.
4. Choose branch `main` and folder `/root`.
5. Save.
6. Your site will publish at a GitHub Pages URL.
